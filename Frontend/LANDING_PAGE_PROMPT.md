## Visual Design System

### **Colors**:
- Primary: Purple (#8B5CF6)
- Secondary: Cyan (#06B6D4)
- Accent: Pink (#EC4899)
- Dark: #0F172A
- Gradients: Purple â†’ Blue â†’ Cyan

### **Typography**:
- Headlines: Inter/Sora (bold, 700-800 weight)
- Body: Inter (regular, 400-500 weight)
- Code/mono: JetBrains Mono

### **3D Effects to Use**:
1. **Glassmorphism** - frosted glass cards
2. **Neumorphism** - soft shadows
3. **Parallax scrolling** - different layer speeds
4. **Perspective transforms** - CSS 3D transforms
5. **Animated gradients** - flowing color transitions
6. **Particle systems** - floating dots/nodes
7. **Mouse tracking** - elements follow cursor
8. **Smooth scrolling** - GSAP/Framer Motion animations
9. **3D hover effects** - cards tilt on hover
10. **Depth shadows** - realistic depth perception

### **Animation Guidelines**:
- Use Framer Motion for React animations
- GSAP for complex timeline animations
- CSS transforms for simple hover effects
- Intersection Observer for scroll animations
- Smooth ease curves (ease-out preferred)
- Animation duration: 0.3-0.6s for UI, 1-2s for complex
